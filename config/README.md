# Config Notes

`config/` stores portable templates and locally rendered runtime files.

- `otel-collector.template.yaml` is committed and contains placeholders.
- `otel-collector.local.yaml` is generated by `python scripts/setup_runtime.py --mode hybrid --render-otel-config`.
- `config/applied/` is created locally when runtime setup writes backups of `~/.openclaw/openclaw.json`.
- Only templates and docs belong in git. Local backups and rendered configs stay ignored.
